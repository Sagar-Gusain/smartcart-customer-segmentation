# SmartCart Customer Segmentation

## Overview

This project various clustering methods to segment customers based on their demographics, purchasing behavior, and engagement data. The goal is to identify meaningful customer groups to support personalized marketing strategies.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Features

- Customer Demographics (Age, Education, Marital Status, Income)
- Purchase Behaviour (Wine, Fruits, Meat, Fish, Sweets, Gold Products)
- Purchase Frequency (Web, Store, Catalog, Deals)
- Customer Engagement (Recency, Website Visits, Complaints)

## Project Workflow

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Data Preprocessing
- Feature Scaling
- K-Means Clustering
- Cluster Analysis
